
# wikipedia syntax


### 1. Text styling


```py
''Italic Text''                 # use 2 single quotation
'''Bold Text'''                  # use 3 single quotation
'''''Bold & Italic'''''         # use 5 single quotation
<code>Inline Code</code>        # create a frame around the text
<s>Strikethrough</s>           # similar to ~~Strikethrough~~ in md (wrong or deprecated text)
<u>Underline</u>               # underline
<blockquote>Quote text</blockquote>      # quote text with a small styling
x<sup>2</sup>                         #  Superscript
H<sub>2</sub>O                       # Subscript
<small>Small Text</small>             # small text
<big>Big Text</big>                    # avoid
- - -                                 # horizontal line
```

### 2. Headings

```py
# never use = Heading =           # do not use 1 =
== Heading ==                    # use 2 =
=== Subheading ===               # use 3 =
==== Details ====                # use 4 =
```


## Links

```py
# internal link
[[Name Of Page]]              # internal link in wikipedia
[[Name of page| text to display]]     # change the name
[[Article#Section]]                # specific section
[[fr:Python]]                      # Interlanguage Link
[[commons:Category:Cats]]                # to another Wikimedia project

# external link
https://www.example.com             # bare URL
[https://www.example.com Custome text]    # External Link with Custom Text
```
